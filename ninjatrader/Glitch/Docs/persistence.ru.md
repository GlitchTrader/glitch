# Хранение данных

## Каталог

Glitch хранит состояние в `GlitchData` внутри пользовательского каталога NinjaTrader, с резервным путём в локальные данные приложения. Сделайте резервную копию перед заменой установки или переносом ПК.

## Модель

Большинство записей — UTF-8 TSV с комментариями `#`; кэш analytics — JSON.

| Файл | Назначение |
|---|---|
| `AccountGroups.tsv` | masters, followers, ratios и маршруты |
| `AccountOverrides.tsv` | ручная классификация |
| `AccountPeaks.tsv` | пики equity для риска |
| `WindowPlacement.tsv` | положение и размер окна |
| `Journal.tsv` | операционные события |
| `CriticalWarnings.tsv` | критические предупреждения |
| `tradeledger.tsv` | round trips из исполнений |
| `risklocks.tsv` | доказательства risk locks |
| `FundamentalCache.tsv` | сохранённый контекст |
| `AnalyticsBridgeCache.json` | данные инструментов/таймфреймов |
| `uisettings.tsv` | UI и язык |
| `RuntimePolicy.tsv` | функции, репликация и риск |
| `LicenseCache.tsv` | защищённый кэш лицензии |
| `Localization.tsv` | локальные переводы |

## Source и runtime

AddOn поставляет значения по умолчанию и шесть языков. `GlitchData` хранит состояние машины и не должен становиться источником defaults. NinjaTrader остаётся авторитетным для счетов, позиций, ордеров и исполнений.

## Восстановление

При обновлении сохраните `GlitchData`. Для переноса копируйте его при закрытом NinjaTrader, затем проверьте группы, счета, ratios, риск, лицензию и ордера перед включением Replication.

## Конфиденциальность

Каталог может содержать идентификаторы счетов, историю, настройки и защищённую лицензию. Резервные копии являются приватными.
